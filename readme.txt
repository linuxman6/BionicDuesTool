■ソフト紹介
　Bionic Dues の出現ターミナルの記憶を補助するツールです。
　また、終了時の状態を保存できるので、時間を空けたプレイ時にも使えるでしょう。

■インストール方法
　任意の場所にアーカイブを展開。

■アンインストール方法
　展開時に出てきたファイルを全削除。
　レジストリや他の場所にファイルは作成していません。
